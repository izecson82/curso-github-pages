Curso API

The API provides up-to-date currency exchange rates from instant transfer services and banks.

Base URLs

Production https://api.curso.best
Test https://test.api.curso.best

Authentication

Send your API key in the request header.

X-API-Key: <your_key>

Endpoints

GET /rates

Returns a list of rates for a currency pair.

ParameterTypeRequiredDescription
fromstringyesThe currency code being converted by the user.
tostringyesThe currency code received by the user.

Headers: X-API-Key, Accept: application/json

Example request (curl) 
curl -H "Authorization: ApiKey YOURAPIKEY" "https://api.curso.best/rates?from=860&to=643"
Example response 
{
  "rates": [
    {
      "service": "Service Name",
      "type": "To card",
      "rate": 140.0,
      "timestamp": "2025-10-07 12:00"
    },
    {
      "service": "Service Name",
      "type": "Exchange",
      "rate": 141.0,
      "timestamp": "2025-10-07 12:00"
    }
  ]
}
Response contents 
service is not a unique identifier of a transfer/bank/payment system.

Possible values of type:
* To card ‚Äî transfer to a bank card by card number or phone
* Cash ‚Äî cash pickup
* App Exchange ‚Äî exchange in a bank‚Äôs mobile app
* Official rate ‚Äî Central Bank rates
* Payment system ‚Äî payment system rate
* App FX Exchange ‚Äî Forex exchange rate in a bank or broker mobile app
The list may be extended.
